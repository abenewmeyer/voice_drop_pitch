VoiceDrop Retention Audit — Darren Allridge
An interactive retention strategy brief built specifically for VoiceDrop's Retention Specialist Consultant application.
What this is
This single-page application replaces a Loom video submission. It demonstrates platform knowledge, a retention diagnostic framework, and the consulting approach I would bring to VoiceDrop's customer success team — built around VoiceDrop's specific market verticals: real estate, financial services, and insurance.
What it includes

Churn impact calculator — interactive model showing the real revenue cost of current churn and the recoverable value of structured CS intervention
Retention diagnostic — four identified failure points specific to VoiceDrop's customer profile, with signal mapping for each
90-day engagement roadmap — phased consulting structure with defined deliverables per phase
Consulting context — background on why this experience set fits this role

Tech
Single HTML file. No dependencies, no build step, no framework. Vanilla HTML, CSS, and JavaScript.
Deployment
Deployed via Vercel. To run locally, just open index.html in any browser.# voice_drop_pitch
